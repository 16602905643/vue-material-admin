# vue-material-admin

基于 Vue2.6x + Vuex + Vue-router + Vuetify2x 实现的后台管理系统模板，改变了传统分页模式使用了上拉加载数据和左右菜单布局增加上左右式布局以支持响应式移动端,
支持颜色主题修改，多语言切换，黑白模式切换

## 功能列表

+ 登录
+ Ecahrt 仪表盘
+ 任务（上拉加载数据和左右菜单布局增加上左右式布局）
+ 简单的流程图编辑
+ 地图点位聚会效果
