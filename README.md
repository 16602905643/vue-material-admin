# vue-material-admin

基于 Vue2.6x + Vuex + Vue-router + Vuetify1.5x 实现的后台管理系统模板，改变了传统分页模式使用了上拉加载数据和左右菜单布局增加上左右式布局以支持响应式移动端。

>感谢 ngx-admin 团队，仪表盘参考了[ngx-admin](http://akveo.com/ngx-admin/#/pages/dashboard) 的风格。

## 开发 && 打包
```
npm install 
 
npm run serve

npm run build
```
